## Front end project boilerplate with react and mobx.


### Technology stack

- React 16
- React-Router 4
- MobX 4
- Webpack 4
- ES6/ES7(async、await)
- Less/Scss
- React Intl
- axios
- socket.io client

### Getting started
#### Install dependencies

```
yarn
```
or
```
npm install
```

### Usage

#### Start project on local server (go to localhost:8083)

```
yarn run dev
```
or
```
npm run dev
```

#### Build project (in "dist" folder):

```
yarn run build
```
or
```
npm run build
```
